\begin{abstract}
% TARGET: ~200 words summarizing problem, method, validation, impact
Reference-based structural annotation of protein-coding genes via orthology is well-established [CITATION TOGA 2023], but extending alignment-based orthology inference to non-coding RNAs remains challenging. Unlike proteins, ncRNAs exhibit low sequence identity across species while maintaining functional structure, requiring structure-aware methods beyond traditional sequence alignment. We present CURIA (Cross-species Unified ncRNA Inference and Annotation), a pipeline that adapts genome alignment chains for ncRNA orthology prediction by leveraging RNA foundation model embeddings as a proxy for structural conservation. CURIA identifies potentially orthologous loci via chain lifting, then uses RNA-FM embeddings to: (1) distinguish structured functional regions ("islands") from genomic background via logistic regression on PCA-transformed embeddings; (2) refine short ncRNA boundaries ($\leq$160 nt) through perturbation-based optimization; and (3) align long ncRNA islands across species using windowed maximum mean discrepancy (MMD) in embedding space. Validation across eight mammalian genomes (human, mouse, dog, cow, horse, pig, rhesus macaque) demonstrates [RESULTS PLACEHOLDER: recovery rates, precision/recall vs. Ensembl]. CURIA provides orthology-aware ncRNA annotations for species lacking comprehensive experimental data, complementing protein-coding gene annotation pipelines.
\end{abstract}

\section{Introduction}

\subsection{Background: Protein-coding gene orthology is solved}
Reference-based annotation of protein-coding genes across species has matured substantially, with methods like TOGA [CITATION TOGA 2023] successfully integrating genome alignments with gene structure prediction to infer orthologs at scale. These approaches leverage high sequence conservation in coding sequences and well-defined open reading frames to reliably project annotations from well-studied reference genomes to query species.

\subsection{The ncRNA orthology challenge}
Non-coding RNAs present a fundamentally different challenge. While ncRNAs play critical roles in gene regulation, chromatin organization, and cellular function [CITATION ncRNA functions], they exhibit:
\begin{itemize}
\item \textbf{Low sequence identity} across even closely related species, often below detection thresholds for standard alignment-based methods
\item \textbf{Structure-function relationships} where secondary/tertiary structure, not primary sequence, determines function [CITATION RNA structure-function]
\item \textbf{Length heterogeneity} spanning 20~nt microRNAs to multi-kilobase long non-coding RNAs (lncRNAs)
\item \textbf{Annotation gaps} in non-model organisms, limiting comparative genomics studies
\end{itemize}

\subsection{Why alignment chains + embeddings?}
Despite low sequence identity, syntenic conservation of ncRNA loci across species [CITATION synteny ncRNA] suggests orthology relationships persist. Genome alignment chains capture these syntenic relationships, providing candidate orthologous regions. However, sequence-based scoring is insufficient for ncRNAs—we need structure-aware similarity metrics.

RNA foundation models like RNA-FM [CITATION RNA-FM] encode both sequence and implicit structural information into embeddings, trained on millions of RNA sequences. These embeddings capture functional conservation missed by sequence alignment alone. By combining alignment-derived loci with embedding-based structural similarity, we can extend orthology inference to ncRNAs.

\subsection{Contributions}
We present CURIA, which:
\begin{enumerate}
\item Adapts chain-based orthology detection (from TOGA) to ncRNA contexts using gene-level exon unions
\item Introduces embedding-based "island scanning" to localize functional elements within noisy lncRNA loci
\item Develops separate pipelines for short ($\leq$160~nt, single-exon) vs. long (multi-exon or $>$160~nt) ncRNAs, optimized for RNA-FM's context-dependent encoding
\item Validates against reference annotations across eight mammalian genomes
\item Provides open-source annotations and pipeline for community use
\end{enumerate}

\section{Methods}

\subsection{Pipeline Overview}
CURIA consists of five major steps:
\begin{enumerate}
\item \textbf{Preprocessing}: Collapse reference isoforms to gene-level exon unions (union transcripts)
\item \textbf{Chain lifting (RNA TOGA)}: Identify candidate orthologous regions in query genome via adapted TOGA algorithm
\item \textbf{Short ncRNA refinement}: For single-exon transcripts $\leq$160~nt, optimize boundaries via perturbations and MMD minimization
\item \textbf{Long ncRNA island detection}: For longer transcripts, identify structured "islands" in both reference and query using embedding-based classification
\item \textbf{Island alignment}: Match reference and query islands via windowed MMD in embedding space
\end{enumerate}

\subsection{Input Data}
\paragraph{Required inputs:}
\begin{itemize}
\item Reference genome annotation (BED12 format) with transcript structures
\item Transcript metadata (gene ID, biotype) from Biomart or equivalent
\item Reference and query genomes (2bit format)
\item Genome alignment chains (e.g., from UCSC, lastz, or minimap2)
\end{itemize}

\paragraph{Preprocessing: Gene-level exon union}
To handle isoform complexity and reduce redundancy, we collapse all isoforms of each gene into a single representative transcript representing the union of all exons across isoforms (gene-level exon union). This ensures comprehensive coverage of potentially functional regions while avoiding redundant processing. Protein-coding genes are included during chain lifting (improving synteny detection) but filtered from final ncRNA outputs.

\subsection{Step 1: Chain-based Orthologous Region Prediction (RNA TOGA)}
We adapt the orthologous region identification component (step 1) from TOGA's pipeline for ncRNAs. Note that TOGA's full workflow includes: (1) annotation vs. chain mapping to find potential orthologous regions, (2) CESAR realignment within those regions, (3) gene loss/pseudogene classification, and (4) final mapping. For CURIA, we use only the initial chain-based region identification, adapted for ncRNA constraints.

\paragraph{Modified chain projection:}
\begin{enumerate}
\item Parse alignment chains to extract syntenic blocks between reference and query genomes
\item For each reference gene-level exon union, project exons through chains to identify query loci
\item Classify projections based on chain coverage and continuity
\item Extract flanking regions ($\pm$X bp; parameter tunable) around projected loci to account for boundary uncertainty
\end{enumerate}

\textbf{Key modifications from protein TOGA:}
\begin{itemize}
\item \textbf{Simplified feature set:} Use only 3 classification features [PLACEHOLDER: specify which features—e.g., chain coverage, exon continuity, syntenic context] instead of TOGA's full model. This reduces false negatives for divergent ncRNAs while maintaining specificity.
\item \textbf{No CESAR realignment:} Skip amino acid-based realignment (not applicable to ncRNAs); boundary refinement handled downstream via embeddings
\item \textbf{No reading frame or splice site scoring:} Not applicable to ncRNAs
\item \textbf{Relaxed sequence identity thresholds:} ncRNAs tolerate higher divergence than protein-coding genes
\item \textbf{Gene-level exon union:} Process collapsed exon sets rather than individual transcript isoforms
\end{itemize}

\textbf{Output:} Candidate orthologous regions in query genome for each reference ncRNA.

\subsection{Step 2a: Short ncRNA Pipeline ($\leq$160 nt, single-exon)}

\paragraph{Rationale:} Short ncRNAs (miRNAs, snoRNAs, tRNAs) fit entirely within RNA-FM's effective context window (~200--400~nt optimal range). However, RNA-FM embeddings are context-dependent—shifting boundaries by even 1~nt changes the embedding. We cannot simply encode once and extract subsequences.

\paragraph{Boundary optimization via perturbations:}
\begin{enumerate}
\item Extract reference ncRNA sequence and candidate query region
\item Generate reference and query embeddings using RNA-FM (mean pooling over sequence length)
\item Compute baseline MMD distance [CITATION MMD] between reference and query embeddings
\item Perturb query boundaries: expand/contract by $\pm$1 to $\pm$N nt (default N=10)
\item Re-encode each perturbation and compute MMD vs. reference
\item Select query boundaries minimizing MMD distance
\end{enumerate}

\paragraph{PCA dimensionality reduction:}
To improve MMD stability and computational efficiency, we apply PCA to RNA-FM embeddings (640 dimensions → K dimensions; K determined by retained variance $>$95\%).

\textbf{Output:} Refined query coordinates and alignment score (MMD distance) for short ncRNAs.

\subsection{Step 2b: Long ncRNA Island Detection}

\paragraph{Motivation:} Long ncRNAs (lncRNAs, long antisense RNAs) span hundreds to thousands of nucleotides, exceeding RNA-FM's optimal context window. Most lncRNA sequence is likely non-functional or lowly constrained. We hypothesize functional elements ("islands") are localized and detectable via deviation from genomic background in embedding space.

\paragraph{Training a signal/noise classifier:}
\begin{enumerate}
\item \textbf{Positive set:} Extract 1,000s of windows from annotated human ncRNAs (exonic regions), encode with RNA-FM, apply PCA, mean-pool to single vector per window
\item \textbf{Negative set:} Extract windows from random intergenic regions + protein-coding exons (structure but different function), encode similarly
\item \textbf{Classifier:} Train logistic regression on PCA-reduced embeddings (2--5 principal components). Achieved ROC-AUC [RESULT PLACEHOLDER] on held-out test set.
\end{enumerate}

\paragraph{Island scanning:}
Apply 72~nt sliding windows with 16~nt stride to:
\begin{itemize}
\item Reference union transcript exons
\item Collapsed query orthologous regions (merged across multiple reference projections to avoid redundant computation)
\end{itemize}

Windows scoring above threshold (e.g., probability $>$0.6) are labeled "structured." Merge adjacent structured windows into contiguous "islands."

\textbf{Key design choice:} 72~nt windows with 16~nt stride balance resolution (detecting small elements) vs. computational cost. Smaller strides risk over-fragmentation; larger windows miss fine-grained structure.

\textbf{Reusability:} Reference islands depend only on reference genome and are precomputed once, cached, and reused across query species.

\subsection{Step 3: Island Alignment via Windowed MMD}

\paragraph{Challenge:} Reference and query islands may differ in length due to indels or annotation boundary differences. Direct MMD comparison requires equal-length inputs.

\paragraph{Windowed alignment:}
\begin{enumerate}
\item Split reference and query islands into 80~nt windows with 4~nt stride
\item Encode each window with RNA-FM (per-nucleotide embeddings, not mean-pooled)
\item For each reference window, compute MMD distance to all query windows in the same orthology group
\item Assign reference-query window pairs minimizing MMD (greedy matching or optimal assignment)
\item Aggregate island-level alignment score as mean MMD across matched windows
\end{enumerate}

\textbf{Stride selection:} 4~nt stride is near the limit where smaller perturbations risk collapsing meaningful structure. Empirically determined via pilot experiments [SUPPLEMENTARY DATA IF NEEDED].

\textbf{Output:} Matched island pairs with alignment scores, providing refined orthologous ncRNA annotations.

\section{Validation}

\subsection{Experimental Setup}

\paragraph{Genome pairs:}
We selected eight mammalian genomes with relatively comprehensive ncRNA annotations:
\begin{itemize}
\item \textbf{Reference:} Human (hg38)
\item \textbf{Query species:} Mouse (mm10, mm39), dog (canFam5), cow (bosTau9), horse (equCab3), pig (susScr11), rhesus macaque (rheMac10)
\end{itemize}

\paragraph{Ground truth:} Ensembl ncRNA annotations (release [VERSION PLACEHOLDER]) for each query species, filtered to orthologous loci (via Ensembl Compara orthology calls where available).

\paragraph{Evaluation metrics:}
\textbf{CRITICAL NUMBERS TO REPORT:}
\begin{enumerate}
\item \textbf{True Positive (TP):} CURIA annotates a region AND Ensembl annotates overlapping ncRNA ($\geq$50\% reciprocal overlap)
\item \textbf{False Negative (FN):} Ensembl annotates ncRNA, CURIA does not annotate overlapping region
\item \textbf{True Negative (TN):} CURIA does not annotate, Ensembl does not annotate (genomic background)
\item \textbf{Sensitivity (Recall):} TP / (TP + FN) — fraction of known ncRNAs recovered
\item \textbf{Precision:} TP / (TP + FP\textsubscript{known}) — but see limitations below
\end{enumerate}

\paragraph{Evaluation challenges:}
\begin{itemize}
\item \textbf{Unknown true negatives:} If CURIA annotates a locus with no Ensembl annotation, we cannot definitively call it a false positive—it may be a true but unannotated ncRNA. We report these as "novel predictions" for manual inspection.
\item \textbf{Annotation heterogeneity:} Ensembl biotype classifications vary; some species have only high-confidence annotations.
\item \textbf{Boundary ambiguity:} ncRNA boundaries are often imprecise; we use 50\% reciprocal overlap threshold.
\end{itemize}

\section{Discussion}

\subsection{Key Findings}
\begin{itemize}
\item Genome alignment chains + RNA foundation model embeddings enable orthology-aware ncRNA annotation despite low sequence identity
\item Separate treatment of short vs. long ncRNAs accommodates context-dependence of RNA-FM
\item Island-based localization addresses the challenge of long, poorly conserved lncRNAs
\item Validation across eight mammals demonstrates [SUMMARY OF PERFORMANCE]
\end{itemize}

\subsection{Comparison to Existing Methods}
[CITATION existing ncRNA annotation tools: Infernal, RNAcode, EvoFold, etc.]
Unlike alignment-free structure prediction (Infernal, RNAfold) or covariance models requiring multiple sequence alignments, CURIA leverages pairwise genome alignments and pre-trained foundation models, enabling:
\begin{itemize}
\item Application to any genome pair with alignment chains (no need for multi-species MSA)
\item Implicit structure awareness via embeddings (no explicit secondary structure modeling)
\item Scalability to thousands of ncRNAs genome-wide
\end{itemize}

\subsection{Limitations}
\begin{enumerate}
\item \textbf{RNA-FM dependence:} Performance tied to RNA-FM's training data and biases; may underperform on RNA classes underrepresented in training
\item \textbf{Validation challenges:} Cannot definitively assess false positive rate due to incomplete reference annotations
\item \textbf{No explicit structure validation:} Currently lacks secondary structure comparison (ViennaRNA integration planned)
\item \textbf{Chain quality dependence:} Poor alignment chains (fragmented, low coverage) limit orthology detection
\item \textbf{Computational cost:} GPU required for practical runtimes; CPU-only mode is slow
\end{enumerate}

\section{Conclusions and Future Directions}

\subsection{Summary}
CURIA demonstrates that alignment-based orthology inference, successfully applied to protein-coding genes, can be extended to non-coding RNAs by substituting sequence similarity with structure-aware embedding similarity. This enables reference-guided ncRNA annotation in non-model organisms.

\subsection{Future Work}
\begin{itemize}
\item \textbf{Secondary structure validation:} Integrate ViennaRNA base-pairing probability matrices (BPPM) to filter predictions lacking structural conservation
\item \textbf{Multi-species MSA:} Extend pairwise comparisons to multi-species alignment for higher confidence
\item \textbf{Functional validation:} Experimentally test novel predictions in cell lines or animal models
\item \textbf{Foundation model fine-tuning:} Train ncRNA-specific embeddings or fine-tune RNA-FM on orthology tasks
\item \textbf{Broader taxonomic coverage:} Validate on non-mammalian clades (birds, reptiles, fish)
\end{itemize}
